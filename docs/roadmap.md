# 개발 로드맵

## 0. 현재 단계 - 골격

- 솔루션과 프로젝트 구성
- 공통 빌드 설정
- MVVM 디렉터리
- Git 제외 규칙

## 1. System Tray

- WPF Application Definition과 Main Window 추가
- 트레이 아이콘 표시
- 창 열기 및 종료 메뉴
- 창을 닫아도 백그라운드 유지 여부 결정

## 2. Clipboard MVP

- 텍스트 변경 감지
- SQLite 저장
- 최근 100개 표시
- 중복 저장 방지
- 선택 항목 다시 복사

## 3. Clipboard 개선

- 실시간 검색
- 고정 / 삭제 / 전체 삭제
- URL, 이메일, 코드형 텍스트 분류
- 민감정보 저장 방지

## 4. Quick Settings

- Bluetooth, Sound, Display, Network 설정 열기
- Downloads 폴더와 Startup Apps 열기

## 5. Audio Status

- 현재 출력 장치 표시
- 기본 장치 변경 감지
- 연결 / 해제 알림
- 트레이 설명 갱신

## 6. 배포

- 앱 아이콘
- 스크린샷과 데모 GIF
- MSIX 또는 exe 설치 방식 확정
- Release 빌드와 설치 검증

## MVP 밖의 범위

클라우드 동기화, 로그인, 모바일 앱, 브라우저 확장, 자동 업데이트는 첫 버전에서 제외합니다.
